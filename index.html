The all-important tech stack

We made this app to showcase the power of using APIs to quickly develop software. It was built over the weekend and certainly has some bugs and features to be desired. Our hope is that you will hack on it or get inspired to create something that awesome using APIs.

Frontend

  • Zurb's Foundation framework
  • jQuery for the ajax goodness
  • Shiny flag icons by IconDrawer

Backend

  • Firebase REST api for the datastore
  • MyMemory translation API for the translations
  • Unirest HTTP library glues it all together
